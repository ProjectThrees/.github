# 🛍️ Student Marketplace

A full-stack marketplace application designed for students to buy and sell items such as textbooks, furniture, electronics, and more. The platform provides a simple and secure way for users to post listings, browse items, and communicate with other students.

---

## 🚀 Features

### Core Features (MVP)
- User authentication with OAuth2 (Google & GitHub)
- Browse and search listings
- Create and manage listings
- View detailed item pages
- Messaging system between buyers and sellers

### Additional Features
- Favorite listings
- Reporting system for users and listings
- Admin moderation (ban, warn, delete users/listings)

---

## 🏗️ Tech Stack

### Frontend
- React (Web)
- React Native (Mobile - Android)

### Backend
- Spring Boot (REST API)
- Spring Security + OAuth2

### Database
- Supabase (PostgreSQL)

### Infrastructure
- Docker
- Render (Deployment)
